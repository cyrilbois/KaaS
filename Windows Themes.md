# Windows Themes

Windows themes are located under:

- User: `%LOCALAPPDATA%\Microsoft\Windows\Themese`
- System: `%WINDIR%\Resources\Themes` (also `%WINDIR%\Resources\Ease of Access Themes`)

Additionally, Cursors are located at: `%WINDIR%\Cursors`.
